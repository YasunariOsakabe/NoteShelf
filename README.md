# NoteShelf
NoteShelfは、読書メモを効率的に管理するためのiOSアプリです。本のメモ保存できる他、メモを整理する機能が豊富に備わっています。
シンプルかつ直感的なデザインで、読書の成果を最大限に活用できます。

# 主要機能
- 読書メモの保存
- ジャンル別管理
- タグ付け
- 検索機能 - 本のタイトル、ジャンル、メモ内容、タグでの検索が可能
- リマインダー機能 - メモの内容を定期的に復習するためのリマインダー機能
- 画像メモ - 読書メモに画像を追加
- メモのエクスポート - メモをPDFやテキストファイルとしてエクスポートし、共有
- ソート機能 - タイトル、ジャンル、追加日でメモをソート
